基于轮廓识别的新型二维码   
============================== 
编译需要qt和opencv 3.2的支持         

生成范例
-----------------------------------
程序中有已经制作好的三个模板，任选其一即可，当然也可以自制模板。                          

四个角上的识别标记可以被替换为其他图形，只要该图形的最深层次大于等于6。
